 Множества:

- s = set() - создание пустого множества
- s = {'a', 'b', 'c'} - не запоминает порядок добавления
- s = {'a', 'a'} == {'a'} - не может содержать одинаковых элементов
- if 'a' in s: ... - проверка на наличие элемента

 Методы:

- s.add(element) - добавление элемента
- s.remove(element) - удаление элемента (с вызовом исключения)
- s.discard(element) - удаление элемента (без вызова исключения)
- s.clear() - удаление всех элементов
