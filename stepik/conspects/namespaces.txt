 Пространство имен:

- пространство имен - множество ссылок от всех имен(переменных, функций и т.п.) до объектов
- builtins - создается при запуске, содержит встроенные функции и классы(int, float, bool, print(), id() ...)
- global - создается при определении переменных в начале кода (внутрь функций не попадает, поэтому их нельзя там использовать)
- function nameSpace - создается при создании функции (в него входят аргументы и локальные переменные, созданные внутри)

- при вызове функции в стеке вызовов порядок nameSpace-ов - (f -> global -> builtins)

 Глобальные переменные:

- обратиться к переменным из nameSpace global можно так - global x = ...
- обратиться к переменным из другого nameSpace кроме global и ниже - nonlocal x = ...
