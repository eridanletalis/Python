 Модули:

- import module - импорт модуля
- module.f(x) - вызов функции модуля
- from module import f - импорт одной функции из модуля (далее функции вызываются просто в коде - f(x))
- from module import* - импорт всех функций модуля
- from module import f as my_f - импорт функции с переименованием

 Принцип действия:

- при использовании import MyClass мы выполняем ВЕСЬ код из файла
- для использования global nameSpace этого класса, мы обращаемся к нему как к атрибуту импортированного файла (MyClass.f(x))
- import происходит единожды и записывается в sys.modules (это словарь {'file_name': file})
- модулями являются не только файлы, но и ПАКЕТЫ (в начале содержат файл __init__.py)

 Полезные методы:

- if __name__ == '__main__': do_smth() else: !do_smth - при импортировании класса код первого блока не будет использован
