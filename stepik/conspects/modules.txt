 Модули:

- import module - импорт модуля
- module.f(x) - вызов функции модуля
- from module import f - импорт одной функции из модуля (далее функции вызываются просто в коде - f(x))
- from module import* - импорт всех функций модуля
- from module import f as my_f - импорт функции с переименованием
