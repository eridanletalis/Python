 Объекты:

- объекты - абстракция(коробка) для данных
- данные - объекты и отношения между ними
- объект - идентификатор + тип + значение

 Неизменяемые(Immutable) объекты:

- числа:
    * int
    * float
    * complex
- bool
- str
- tuple (кортежи, т.е. неизменяемые списки) *можно использовать в качестве ключа в dict
- frozenset - неизменяемое множество

 Изменяемые(Mutable) объекты:

- list
- set
- dict
