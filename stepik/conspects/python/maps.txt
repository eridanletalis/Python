 Словари(ассоциативные массивы):

- d = dict() / {} - создание пустого словаря
- d = {'a': 1, 'b': 2} - создание словаря
- d['a'] - доступ к значению по ключу
- d['a'] = 3 - изменяет значение по данному ключу, либо добавляет новую пару ключ-значение
- ключи УНИКАЛЬНЫ и НЕИЗМЕНЯЕМЫ! (могут быть int и str в разных парах)
- значения ЛЮБОГО ТИПА!

 Операции:

- if key in (not in) d: ... - проверка наличия ключа
- d.get(key) - возвращает либо значение, либо None
- del d[key] - удаление пары из словаря
- d.keys() - множество ключей
- d.values() - множество значений
- for key, value in d.items(): ... - перебор всего словаря с парами ключ-значение
